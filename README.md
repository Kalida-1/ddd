## Installation

### Requirements
- Node.js 16+ or Python 3.8+
- 4GB RAM
- Modern operating system

### Setup

```bash
git clone https://github.com/dev/website-analytics-pro.git
cd website-analytics-pro
npm install  # or: pip install -r requirements.txt
npm start
```
